Supabase Setup Guide

1. Create a Supabase project — Go to supabase.com, sign up free, and create a new project. Wait for it to finish setting up (~2 min).

2. Run the SQL below — In your project go to SQL Editor → New Query, paste the SQL below and click Run. This creates all the tables.

3. Get your credentials — Go to Settings → API. Copy your Project URL and anon public key.

4. Paste in U-Tech-POS — Enter both values in the setup screen and click Connect.

5. Deploy on Vercel — Upload the HTML file to a GitHub repo, connect it on vercel.com and deploy. Done!
